League Of Legends

Projet React/Vite avec quiz intégré.

Présentation du projet

Ce projet React permet d’afficher des informations sur les champions de League of Legends, avec un quiz intégré et une interface moderne. Les données proviennent de l’API Riot.

Objectifs principaux :

  • Afficher les champions via une API
  • Créer un quiz interactif
  • Utiliser React et Vite
  • Proposer une interface moderne et fluide

Techniques utilisées :

  • React
  • JavaScript
  • Vite
  • API Riot
