Escape Game

Site interactif avec réservation et paiement sécurisé.

Présentation du projet

Ce projet propose un site complet d’Escape Game avec réservation, paiement sécurisé, gestion des salles et interface administrateur.

Objectifs principaux :

  • Créer un système de réservation
  • Gérer les paiements sécurisés
  • Créer un espace administrateur
  • Gérer les salles et les créneaux

Techniques utilisées

  • HTML
  • CSS
  • JavaScript
  • React
  • MySQL
