Graines de Santé

Projet de fin d’étude – En cours de développement

Présentation du projet

Graines de Santé est une application web en cours de développement dans le cadre de mon projet de fin d’étude. L’objectif est de proposer une plateforme moderne permettant de gérer, consulter et mettre en avant des produits alimentaires sains.

Objectifs principaux :

  • Créer une interface moderne et responsive
  • Mettre en place un système de gestion des produits
  • Ajouter des filtres, catégories et recherche
  • Préparer une base solide pour une future API
  • Développer une architecture claire et évolutive

Technologies utilisées

  • HTML
  • CSS
  • JavaScript
  • React
  • Node.js
  • MySQL
