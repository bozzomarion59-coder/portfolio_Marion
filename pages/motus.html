Motus

Jeu interactif inspiré du célèbre Motus.

Présentation du projet

Ce projet est un jeu Motus développé en JavaScript. Il reprend le principe du jeu télévisé : deviner un mot en un nombre limité d’essais, avec des lettres bien placées et mal placées.

Objectifs principaux :

  • Créer une interface dynamique
  • Gérer la logique du jeu (lettres bien/mal placées)
  • Afficher les résultats en temps réel
  • Proposer une expérience ludique et fluide

Techniques utilisées

  • HTML
  • CSS
  • JavaScript
