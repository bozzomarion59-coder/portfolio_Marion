PORFOLIO - MARION BOZZO

# Description
Ce projet est mon **portfolio personnel** réalisé uniquement avec **HTML, CSS et JavaScript**.  
Il présente mon parcours, mes compétences techniques et mes projets réalisés.  
L’objectif est de proposer une interface claire, moderne et interactive aux recruteurs et visiteurs.


# Technologies utilisées
- HTML pour la structure du site.  
- CSS pour le design, la mise en page et le responsive.  
- JavaScript pour rendre le site dynamique et interactif.  



# Pourquoi avoir utilisé du JavaScript ?
J’ai choisi d’intégrer JavaScript dans mon portfolio car il permet à mes boutons "vers GitHub" de créer un évenement au clic qui redirige directement vers la page GitHub avec un effet de couleur au passage de la souris. 


# Projets inclus
- CV en ligne : Création d’un CV responsive en HTML/CSS.  
- Jeu Motus : Jeu interactif en JavaScript inspiré du Motus.  
- Gestion des utilisateurs : Application en HTML/CSS/JS/React simulant la gestion d’utilisateurs.
